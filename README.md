# 🧠 Habit Tracker (Rastreador de Hábitos)

Um aplicativo simples e moderno para controle de hábitos diários, desenvolvido com React + TypeScript.

## 🚀 Objetivo

Este projeto foi criado com o objetivo de praticar e consolidar conceitos fundamentais do React, como:

* Manipulação de estado (`useState`)
* Componentização
* Props
* Renderização condicional
* Listas e eventos
* Boas práticas de organização de código

---

## ✨ Funcionalidades

* ➕ Adicionar novos hábitos
* 🗑️ Remover hábitos
* ✅ Marcar hábitos como concluídos
* 📊 Contador de progresso (concluídos / total)
* 🎯 Sistema de prioridade (fácil, médio, difícil)
* 🎨 Interface clean e responsiva (mobile-first)

---

## 🛠️ Tecnologias utilizadas

* React
* TypeScript
* Tailwind CSS

---

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto, foram reforçados conceitos como:

* Imutabilidade no React
* Separação de responsabilidades entre componentes
* Gerenciamento de estado centralizado
* Boas práticas de UI/UX
